== Description ==

= Admin =
Update notifications.
Footer text.
Include JS and CSS.
Sidebar.
Posts screen.
Edit/Add Post screen.
Dashboard.
Nav menus.
User Edit/Profile.

= Frontend =
Remove header metas.
Author archive.
Taxonomy archive.
Date archive.

= Debug =
Current Information.
My WP Settings.
Helping debug information.

== Installation ==

1. Upload the entire mywp folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You will find 'My WP' menu in your WordPress admin panel.
